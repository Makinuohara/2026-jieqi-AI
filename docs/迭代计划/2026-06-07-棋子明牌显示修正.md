# 棋子明牌显示修正

## 目标

统一 JavaFX 棋盘的棋子文字显示，确保明牌棋子始终在当前格显示真实棋名。

## 显示规则

- 空棋位不显示文字。
- 未翻开的棋子显示“暗”。
- 初始将帅直接显示“帅”和“将”。
- 暗子第一次合法移动后，在落点显示服务端或本地规则引擎实际翻出的棋名。
- 红方显示“帅、仕、相、兵、车、马、炮”。
- 黑方显示“将、士、象、卒、车、马、炮”。

## 实现

- 新增统一棋名格式化组件，棋盘不再自行维护棋名映射。
- 增加空格、暗子、双方全部棋种和“虚拟兵翻出仕”的自动化测试。
- 将本地辅助目录 `.superpowers/` 加入 `.gitignore`。
- 新增 `.dockerignore`，排除构建产物、版本库和本地辅助目录。
- 修复全局按钮内边距挤压棋格文字导致棋名显示为省略号的问题。
- 走法记录支持自动换行并缩短坐标文字，确保“翻出某棋”直接可见。
- 扩大对局窗口和记录栏的初始尺寸，避免三栏布局互相挤压。
- 走法记录按阵营显示“帅/将、仕/士、相/象、兵/卒”，与棋盘文字保持一致。

## 验收

- 初始棋盘中央将帅位置分别显示“帅”和“将”。
- 任意暗子移动翻开后，原位置为空，目标位置显示实际棋名。
- 红黑双方差异棋名符合中文象棋习惯。
