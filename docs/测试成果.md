# 测试成果

本文档记录当前阶段已经完成的主要验证结果，重点覆盖本地对局、GUI 交互和界面改进。

## 自动化测试

- 使用命令 `sh mvnw -q -pl jieqi-gui test` 验证 GUI 模块通过。
- 使用命令 `sh mvnw test -q` 验证整仓模块测试通过。
- 当前自动化测试共 80 项，覆盖模型、规则、AI、协议、服务器以及本地 GUI 对局。

## 本地对局验证

- 真人对搜索 AI 模式可正常启动、选子、走子、认输和重新开始。
- 本地 AI 对 AI 模式可正常启动，支持自动轮流走子、暂停、继续、单步和速度调整。
- 两种本地模式都能正确显示走法记录与终局状态。

## 界面改进验证

- 棋盘已改为更接近象棋盘的绘制方式，包含九宫、楚河汉界和兵炮标记点。
- 棋子已调整为圆形棋子样式，暗子与明子显示效果可区分。
- 选中棋子的可走项区分为空位落点提示和吃子落点提示。
- 上一步走子的起点与落点会在棋盘上持续高亮，便于观察 AI 行为。
- 左侧信息区新增常驻“将军提示”，能够持续显示红方被将军、黑方被将军、双方对将或当前无将军。
- 被将军一方的将/帅在棋盘上会显示红色危险高亮。

## 启动验证

- 使用命令 `sh mvnw -q -pl jieqi-gui -am install -DskipTests` 后，
  执行 `sh mvnw -q -f jieqi-gui/pom.xml javafx:run`，GUI 可正常启动。
- 使用命令 `sh mvnw -q -pl jieqi-server -am package -DskipTests` 后，
  可正常生成服务器可执行 jar。
